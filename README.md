# ApnaParyawaran 🌱

A modern, responsive, eco-themed social media community platform where people connect, share, and inspire environmental action.

---

## 🌍 Project Mission
Build an engaging social space for ideas, discussion, and action in sustainability, nature protection, and eco-innovation.

---

## 🚀 Features
- **User Auth:** Email & social login, user profiles
- **Social Feed:** Text/image/video posts, likes, comments, shares, follows, trending, hashtags
- **Ideas Sharing:** Innovative ideas, upvote/downvote, discussion
- **Groups & Forums:** Topic communities (Recycling, Renewable Energy, etc.)
- **Events & Map:** Organize/join cleanups, plantation drives, volunteer, live eco-events map
- **Eco Challenges:** Daily/weekly challenges, leaderboard, achievement badges
- **Nature Explorer:** Gallery of beautiful natural places, interactive world map
- **Profile Dashboard:** Stats, eco level badge, contribution charts, saved/drafts
- **AI Features:** Idea generator, chatbot, post summarizer (OpenAI/Google AI)
- **Monetization:** AdSense, donation button, sponsored campaigns, premium membership

---

## 🛠️ Tech Stack
- **Frontend:** React / Next.js
- **Backend:** Node.js
- **Database:** MongoDB or PostgreSQL
- **Auth:** NextAuth.js
- **Cloud Storage:** Cloudinary or S3
- **Realtime:** Socket.io or Pusher
- **Maps:** Google Maps API

---

## 🌱 Getting Started
1. Clone the repository
   ```sh
   git clone https://github.com/Utjha5050/apna-paryawaran.git
   ```
2. Install dependencies (Monorepo suggested)
   ```sh
   cd apna-paryawaran
   npm install
   ```
3. Set up environment variables (see `.env.example`)
4. Run development server
   ```sh
   npm run dev
   ```
